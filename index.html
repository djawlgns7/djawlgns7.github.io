1. HouseKeeping (2024.07.08 ~ 2024.08.25)

프로젝트 소개

HouseKeeping은 혼자 자취하는 사회 초년생들을 위한 사이트입니다.
사용자의 의식주를 전부 관리해주는 기능이 있습니다.
방 청소 루틴을 설정하고 이를 어길 경우 알림 메시지가 오고 내 방의 오염도가 증가하는 기능,
내 냉장고 안에 있는 식재료를 관리할 수 있고 현재 냉장고 안에 있는 식재료로 할 수 있는 요리를 추천해 주는 기능,
옷장 안에 있는 옷을 등록하여 한눈에 보면서 관리할 수 있고 현재 날씨를 고려해서 어떤 옷을 입을 지 추천하는 기능 등이 있습니다.

개발 환경 및 툴

내가 기여한 부분

클라우드 서비스를 활용한 프로젝트 호스팅

  • 네이버 클라우드를 활용하여 프로젝트를 호스팅 했습니다.
  • 하나의 VPC를 용도에 따라 여러 서브넷으로 구분하였고, DB와 소켓 서버와 같은 외부의 접근이 필요하지 않은 서버의 경우 외부의 접근을 차단하여 보안성을 증대시켰습니다.
  • 로드 밸런서를 활용하여 트래픽을 분산시켜서 안정적인 서비스를 제공하게 하였습니다.
  • 도커를 활용하여 서버 환경이 변하더라도 동일한 방식으로 서비스를 제공할 수 있게 하였습니다.
  • Jenkins를 활용하여 CI/CD 파이프라인을 구축하여 지속적인 통합과 배포를 가능하게 하였습니다.
  • SSL VPN을 통해 통신을 암호화하여 보안성을 증대시켰습니다.

소켓을 활용한 채팅 및 온라인 친구 확인 기능 개발

  • 소켓IO를 활용하기 위한 Node.js 서버를 제작하여 안정적인 소켓 통신 기능을 구현했습니다.
  • 1대1 채팅 및 그룹 채팅 기능을 구현했습니다. 수신한 메시지를 팝업 형식으로 표시하는 기능 또한 구현했습니다.
  • 소켓 서버와의 연결을 통해 실시간 접속 상태를 DB에 업데이트 하고, 친구들에게도 알려주는 기능을 구현했습니다.

Three.js를 활용한 방 꾸미기 기능 개발

  • 벽지 및 바닥의 색을 바꿀 수 있고, 다양한 가구들을 배치해서 나만의 방을 꾸밀 수 있는 기능을 구현했습니다.
  • 가구는 자유롭게 이동, 회전 및 크기 조절이 가능하게 하였습니다.
  • 친구의 방을 방문하여, 친구가 꾸민 방을 구경하는 기능을 구현했습니다.

AI를 활용한 레시피 추천 기능 개발

  • 냉장고에 식재료를 등록하는 등록하는 기능을 구현했습니다.
  • 식재료에 메모 및 유통기한을 등록할 수 있게 하였습니다.
  • 유통기한이 임박한 식재료가 있을 경우 CoolSMS API를 활용하여 경고 메시지를 보내는 기능을 구현했습니다.
  • Hyper CLOVA API를 활용하여, 등록된 식재료를 기반으로 음식 레시피를 추천하는 기능을 구현했습니다.
  • 레시피 추천의 경우, 요리의 분류, 요리의 양과 같은 세부 사항을 지정할 수 있게 하였습니다.

데이터베이스 설계 및 구축

  • 요구사항 정의서를 기반으로 ERD 다이어그램을 만들었습니다.
  • ERD 다이어그램을 기반으로 Spring Data JPA를 활용하여 데이터베이스를 구축하였습니다.





2. Q2Beat (2024.06.03 ~ 2024.07.05)

프로젝트 소개

핸드폰만 가지고 있다면 누구나 참여 가능한 퀴즈 게임입니다.
호스트가 방을 만든 뒤, QR 코드 혹은 url을 활용하여 핸드폰으로 방에 입장을 하고, 다같이 퀴즈를 풀어서 순위를 가릴 수 있습니다.
단순한 퀴즈 뿐만 아니라 음성 인식과 문자열 비교 알고리즘을 활용한 잰말놀이 퀴즈,
Machine Leaning을 활용한 모델이 두 사진의 유사도를 측정하여 주어진 포즈와 얼마나 비슷한 포즈를 취했는지 볼 수 있는 포토제닉 퀴즈 등이 있습니다.

내가 기여한 부분

Vercel 및 AWS를 활용한 프로젝트 호스팅

  • 프론트 앤드 서버의 경우 Vercel을 활용하여 호스팅을 하였고, 백앤드 서버의 경우 AWS를 활용하여 프로젝트를 호스팅 했습니다.
  • Certbot을 이용하여 SSL 인증서를 발급받아서 보안성을 증대시켰습니다.

웹 소켓을 활용한 실시간 통신 기능 개발

  • SocketJS를 이용하여 호스트와 참여자 간의 실시간 통신 기능을 구현했습니다.

방 생성 및 참여 기능 개발

  • 호스트가 방을 생성할 경우 고유한 방의 번호를 배정받고, 호스트 및 참여자의 정보를 DB에 저장하는 기능을 개발했습니다.
  • QR 코드 및 URL을 통하여 호스트가 생성한 방에 입장을 할 수 있게 하는 기능을 개발했습니다.
  • 호스트는 퀴즈의 종류 및 주제, 라운드 수, 제한 시간 등을 설정할 수 있고, 게임을 시작할 수 있게 하는 기능을 개발했습니다.
  • 참여자의 경우 닉네임을 설정할 수 있게 하였습니다.

4지선다 퀴즈 개발

  • 상식, 스포츠, IT 지식 등의 주제에 맞는 4지선다 퀴즈를 만들었습니다.
  • 호스트 화면의 경우 문제 및 보기가 표시되고, 제한 시간이 끝날 경우 자동으로 정답이 표시되고 다음 라운드로 넘어가게 했습니다.
  • 참여자 화면의 경우 정답을 선택할 수 있는 버튼이 표시되고, 라운드가 끝날 경우 정답 여부와 현재 내 점수가 표시되게 했습니다.
  • 모든 라운드가 끝날 경우 호스트의 화면에는 참여자들의 순위가 표시되게 했고, 참여자들의 화면에는 등수가 표시되게 했습니다.

잰말놀이(발음이 어려운 문장 따라하기) 퀴즈 개발

  • 발음이 어려운 문장을 보고, 제한 시간 내에 정확하게 따라하는 퀴즈를 만들었습니다.
  • 네이버 클라우드의 CLOVA CSR을 활용하여 참여자의 음성 파일을 텍스트로 변환하는 기능을 구현했습니다.
  • Levenstein 알고리즘을 활용하여 정답 문장과 참여자의 음성을 변환한 문장 사이의 거리(유사도)를 측정하는 기능을 구현했습니다.
  • 참여자의 음성을 wav 파일로 저장하는 기능을 구현했습니다.

동작 따라하기 퀴즈 개발

  • 예시 포즈 이미지를 보고 최대한 비슷하게 따라하는 퀴즈를 만들었습니다.
  • Teachable Machine을 활용하여 다양한 포즈 데이터들을 학습시킨 결과 이미지 파일을 인식하여 특정 포즈와의 유사도를 측정해주는 AI를 만들었습니다.
  • 리액트 웹캠을 활용하여 타이머가 끝날 경우 참여자의 포즈를 찍어 이미지 파일로 저장하는 기능을 구현했습니다.


3. JavaGym (2024.03.25 ~ 2024.04.22)

프로젝트 소개

헬스장을 이용하는 사람들을 위한 키오스크 프로그램 입니다.
회원들은 로그인만 하면 헬스장 입장, PT 예약 등의 기능을 이용 가능합니다.
트레이너는 자신이 담당하는 회원들의 목록과 예약 목록을 볼 수 있습니다. 예약 등록 및 수정, 삭제 또한 가능합니다.
관리자는 헬스장에 등록된 트레이너와 회원들의 정보 열람 및 수정이 가능합니다. 수동으로 트레이너와 회원을 등록하는것 또한 가능합니다. 그리고 회원들의 출입 로그를 확인할 수 있습니다.

내가 기여한 부분

회원들을 위한 기능 개발

  • 회원가입 및 로그인 기능을 구현했습니다.
  • 메인 페이지에서는 현재 사용 가능한 PT 이용권의 갯수, 헬스장 이용권의 남은 일수를 확인할 수 있게 했습니다.
  • 내 정보 페이지에서 회원 정보와 현재 사용중인 사물함 번호, 담당 트레이너 등을 확인하는 기능을 구현했습니다.
  • PT 예약 가능 시간을 확인하고 예약할 수 있는 기능을 구현했습니다.
  • PT 이용권, 헬스장 이용권, 사물함 이용권 등을 구매할 수 있는 결제 페이지를 구현했습니다.
  • 결제 페이지에서는 헬스장에 등록된 트레이너들을 확인하고, 담당 트레이너로 지정할 수 있는 기능을 구현했습니다.
  • 결제 페이지에서는 현재 이용 가능한 사물함 목록을 볼 수 있고, 선택할 수 있는 기능을 구현했습니다.

DB 구축 및 스토어드 프로시저와 이벤트 스케쥴러 개발

  • 요구사항을 기반으로 데이터베이스를 구축했습니다.
  • 다양한 스토어드 프로시저를 만들어 복잡한 쿼리나 코드를 작성하지 않고도, 간단한 로직을 여러 곳에서 반복적으로 활용할 수 있도록 구현하였습니다.
  • 이벤트 스케쥴러를 만들어 매일 회원들의 이용권의 남은 일수를 자동으로 차감하는 기능을 구현했습니다.
