요구사항 정의
## 회원 - 회원가입 및 로그인 할 수 있다 - 회원가입 시 본인 인증을 할 수 있다 - 본인인증 수단은 핸드폰, 구글, 네이버 등 다양하게 존재한다 ## 방 생성(회원) - 로그인을 한 회원은 놀이 방을 만들 수 있다 - 놀이 방에 있는 qr코드를 이용해서 참여자는 방에 들어올 수 있다 - 방장은 원하는 종류의 놀이를 선택할 수 있다 - 이 때 방의 설정을 변경할 수 있다 ## 참여하기 - qr코드를 통해 호스트가 만든 방에 입장이 가능하다(핸드폰 이용) - 처음 qr코드를 통해 방에 들어오면 닉네임을 정해야 한다 - 참여자는 닉네임을 정한 뒤 게임이 시작하기 전까지 대기한다 --- ## 퀴즈🙋🏻‍♂️(호스트) - 퀴즈 게임의 경우 개인전으로 진행된다 - 방장은 게임 시작 전 라운드 수, 제한 시간, 카테고리를 설정할 수 있다 - 카테고리의 경우 상식, 국기 맞추기, 프로그래밍 관련 등 다양하게 존재한다 - 게임이 시작한 뒤 매 라운드마다 문제와 선택지가 방장의 화면에 나타난다 - 제한 시간이 끝난 뒤 문제의 답과 해당 답을 선택한 인원의 수가 화면에 나타난다 - 모든 라운드가 끝난 뒤 퀴즈를 맞춘 갯수에 따라 순위를 정하고, 순위를 화면에 표시한다 --- - VS게임 : (유튜브 공식계정 기준) 뮤직비디오 조회수 대결 - 노래 1초듣고 제목 맞추기 - 노래 가사만 보고 제목 맞추기 ## 퀴즈 👥(참여자) - 게임이 시작될 경우 방장의 화면에 출력된 문제를 보고 올바른 답을 선택해야 한다 - 제한 시간이 끝난 뒤 자신이 선택한 답이 정답인지 여부와 내 점수가 화면에 표시된다 - 모든 라운드가 끝나면 내 등수가 화면에 표시된다 ## 노래 따라부르기🙋🏻‍♂️(호스트) - 노래 따라부르기 게임의 경우 두 팀으로 나누어져 진행된다 - 라운드가 시작되면 부를 노래를 재생시켜 준다 - 그 후 각 팀에서 팀원 한명씩 랜덤으로 노래를 부를 사람이 지정된다 - 해당 라운드가 끝나면 각각 팀의 점수를 화면에 표시해준다 - 모든 라운드가 끝나면 각각 팀의 최종 점수(평균 점수)를 화면에 표시해준다 ## 노래 따라부르기👥(참여자) - 각각 팀의 지정된 사람은 한명씩 차례대로 주어진 소절을 부른다 - 노래가 나오다가 자신의 차례가 되면은 화면의 마이크 버튼을 눌러서 노래를 녹음한다 - 해당 라운드가 끝나면 내 팀의 점수가 화면에 표시된다 - 모든 라운드가 끝나면 각각 팀의 최종 점수가 화면에 표시되고, 승, 패 여부도 같이 표시된다 ## 가사 맞추기 게임🙋🏻‍♂️(호스트) - 가사 맞추기 게임의 경우 개인전으로 진행된다 - 라운드가 시작되면 빈 칸이 있는 가사가 문제로 출제된다(음악은 재생하지 않음) - 제한 시간의 1/4이 지날 때마다 새로운 힌트를 공개한다 - 힌트는 총 3개고, 나중에 나오는 힌트일수록 쉬운 힌트이다 - 첫번째 힌트는 노래 제목 - 두번째 힌트는 초성 공개 - 세번째 힌트는 글자 절반 공개 - 참여자가 오답을 제출할 경우 오답은 화면의 하단에 표시된다 - 참여자가 정답을 맞출 경우 라운드는 즉시 종료되고, 정답과 정답자의 이름이 화면에 표시된다 - 모든 라운드가 끝난 뒤 퀴즈를 맞춘 갯수에 따라 순위를 정하고, 순위를 화면에 표시한다 ## 가사 맞추기 게임👥(참여자) - 게임이 시작될 경우 화면에 표시된 문제를 보고 빈 칸에 들어갈 가사를 입력한다 - 문제를 맞출 기회는 여러번 존재한다 - 해당 라운드가 끝나면 정답 여부와 정답, 그리고 내 점수가 화면에 표시된다 - 모든 라운드가 끝나면 내 등수가 화면에 표시된다 --- ## ⭐️포즈 따라하기🙋🏻‍♂️(호스트) - 포즈 따라하기 게임의 경우 두 팀으로 나누어져 진행된다 - 라운드가 시작되면 따라할 포즈를 화면에 출력한다 - 그 후 각 팀에서 팀원 한명씩 랜덤으로 포즈를 따라할 사람이 지정된다 - 지정된 사람은 제한 시간 내에 지정된 포즈를 따라해야 한다(핸드폰의 전면 카메라 사용) - 해당 라운드가 끝나면 각각 팀의 점수와 팀원이 취한 포즈를 화면에 표시해준다 - 모든 라운드가 끝나면 각각 팀의 최종 점수(평균 점수)를 화면에 표시해준다 - teachable machine의 모션 인식 기능을 활용하여 유사도를 측정하여 점수를 매긴다 ## ⭐️포즈 따라하기👥(참여자) - 라운드가 시작되면 포즈를 취할 사람을 랜덤으로 지정해서 안내해준다(각 팀당 한명) - 지정된 사람들은 제한 시간 내에 지정된 포즈를 취해야 된다 - 해당 라운드가 끝나면 내 팀의 점수가 화면에 표시된다 - 모든 라운드가 끝나면 승, 패 여부가 화면에 표시된다

## 회원(member) - 회원 번호(member_id) int(4) primary key auto_increment (0001~9999) - 아이디(member_username) varchar(20) unique not null - 비밀번호(member_password) varchar(60) not null - 이름(member_name) varchar(20) not null - 성별(member_gender) enum(’Male’, ‘Female’) not null - 생년월일(member_birthday) date not null - 휴대폰 번호(member_phone) int(8) not null - 가입일(member_enrollment) date default (current_date) not null ## 퀴즈(일반)(quiz_normal) - 퀴즈 번호(normal_id) primary key auto_increment (10000~19로999) - 카테고리(normal_category) - 퀴즈(normal_quiz) - 이미지(normal_image) - 1번 선택지(normal_first_choice) - 2번 선택지(normal_second_choice) - 3번 선택지(normal_third_choice) - 4번 선택지(normal_fourth_choice) - 정답(normal_answer) ## 퀴즈(노래 따라부르기)(quiz_singing) - 퀴즈 번호(singing_id) primary key auto_increment (20000~29999) - 장르(singing_category) - 문제 노래(유튜브 링크)(singing_quiz) - 정답 노래(singing_answer) ## 퀴즈(가사 맞추기)(quiz_lyric) - 퀴즈 번호(lyric_id) primary key auto_increment (30000~39999) - 문제 가사(lyric_quiz) - 정답 가사(lyric_answer) - 힌트1(lyric_first_hint) - 힌트2(lyric_second_hint) - 힌트3(lyric_third_hint) --- ## 퀴즈(포즈 따라하기)(quiz_pose) - 퀴즈 번호(pose_id) primary key auto_increment (40000~49999) - 문제 포즈(pose_quiz)
